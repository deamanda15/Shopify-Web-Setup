# Setting Hard-Selling Pop-ups in Shopify

This guide explains how to increase sales and capture leads by setting up high-converting promotional pop-ups in your Shopify store using the **Pop Convert** app.

## Steps to Configure Pop-ups:

### 1. Install Pop Convert App
- Go to the Shopify App Store and install **Pop Convert - Pop Ups, Banners**.
- Once installed, access the app dashboard via Shopify Admin > **Apps** > **Pop Convert**.

### 2. Navigate to the Popups Dashboard
- In the left sidebar of the Pop Convert panel, look under the **INCREASE SALES** section and click on **Popups**.
- From here, you can view existing popup campaigns, see their active status, and track analytical performance (Views, Clicks, and Subscribers).

![Pop Convert App Dashboard](assets/images/setting%20pop-up%20website%20-%20pict%201.png)

### 3. Create a New Popup Campaign
- Click the **Create Popup** button in the top right corner.
- You can design several types of popup campaigns:
  *   **Lead Capture (e.g., "Join Distributor")**: Set up a custom form with input fields for full name, phone number, and a strong CTA (like "Join now!") to recruit distributors or wholesale buyers.
  *   **Newsletter Signups**: Create popups offering subscriptions to update customers on upcoming sales and product drops.
  *   **Discount Offers (e.g., "-10% Off First Order")**: Create high-incentive popups that trigger coupons when users load the page or try to exit.

### 4. Configure Trigger Rules and Targeting (AB Testing)
- Set rules for when the popup should appear:
  *   **Time-delay**: Show the popup after a customer spends a certain number of seconds on your website.
  *   **Exit-intent**: Show the popup right before a customer moves their cursor to close the browser tab.
  *   **Scroll-trigger**: Trigger the popup after a user scrolls through a specific percentage of the homepage.
- Leverage the **AB Testing** tab to test different headlines, designs, or coupon codes to optimize conversion rates.
- Toggle campaigns between **Active** and **Paused** states to manage active promotions.
