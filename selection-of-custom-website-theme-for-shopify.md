# Selecting a Custom Website Theme for Shopify

This guide describes how to select, import, and customize a website theme in Shopify for your online store (in this project, using the theme customizer for `dafanyascarf.id`).

## Steps to Manage and Choose Themes:

### 1. Access the Themes Settings
- Log in to your Shopify Admin.
- In the left sidebar, navigate to **Sales channels** > **Online Store** > **Themes**.
- Here, you will see your current active theme (e.g., **Dawn**), along with details of when it was last saved.

![Shopify Themes Dashboard](assets/images/selection%20of%20custimize-pict%201.png)

### 2. Exploring Popular Free Themes
- Scroll down the Themes page to see the **Popular free themes** section.
- Shopify provides pre-designed free themes like **Dawn**, **Spotlight**, **Craft**, **Sense**, and **Refresh**. You can add any of these to your library by clicking **Add**.
- To see more themes (both free and paid), click on the **Visit Theme Store** button.

![Popular Free Themes](assets/images/selection%20of%20custimize-pict%202.png)

### 3. Importing a Custom Theme (Zip File or GitHub)
- If you have purchased a custom theme or developed one, you can import it:
  - Click on the **Import theme** button at the top right of the Themes dashboard.
  - Choose one of the following methods:
    *   **Visit Theme Store**: Browse official Shopify themes.
    *   **Upload zip file**: Upload a theme file (e.g., from third-party marketplaces like Themeforest).
    *   **Connect from GitHub**: Connect directly to a GitHub repository containing your theme's code.

![Import Custom Theme Options](assets/images/selection%20of%20custimize-pict%203.png)

### 4. Customizing the Theme
- Once your theme is installed, click **Customize** next to the theme name to open the Shopify Theme Customizer.
- Within the customizer, you can configure your layout:
  - **Left Pane (Sections)**: Add or rearrange sections like the Header, Announcement Bar (e.g., adding promotional text like `Use code: DFNWEB to get a discount of up to 45%!`), Image Banner, and Footer.
  - **Middle Pane (Live Preview)**: See changes in real-time. For example, positioning the brand logo (`DAFANYA`), setting up navigation menus, or adding promotional banner images.
  - **Right Pane (Settings)**: Adjust specific section attributes, such as "Desktop logo position" (Top center), "Sticky header" (On scroll up), color schemes, and mobile layout options.

![Shopify Theme Customizer](assets/images/selection%20of%20custimize-pict%204.png)
